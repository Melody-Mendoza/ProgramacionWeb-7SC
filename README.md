# Programación Web - Actividad 1 | Tema 2

## Descripción

Este repositorio reúne los **18 ejercicios desarrollados durante las actividades del Tema 2** de la materia de **Programación Web**. Cada práctica permitió reforzar los conocimientos adquiridos en clase mediante el desarrollo de páginas web sencillas, aplicando los fundamentos de **HTML**, **CSS** y **JavaScript**.

A lo largo de los ejercicios se trabajó de manera progresiva, comenzando con la estructura básica de una página web hasta el desarrollo de aplicaciones interactivas utilizando JavaScript.

---

## Ejercicios realizados

### Ejercicio 1. Hola Mundo
Creación de una página web básica utilizando la estructura principal de un documento HTML e incorporando títulos y párrafos.

### Ejercicio 2. Párrafos y Formato Básico
Uso de diferentes etiquetas para dar formato al contenido, aplicando negritas, cursivas y saltos de línea para mejorar la presentación del texto.

### Ejercicio 3. Enlaces Simples
Creación de hipervínculos que permiten navegar hacia otras páginas web y abrir enlaces en una nueva pestaña.

### Ejercicio 4. Imágenes Básicas
Inserción de imágenes dentro de una página web utilizando rutas locales, texto alternativo y diferentes tamaños.

### Ejercicio 5. Combinando Texto y Enlaces
Desarrollo de una página tipo blog donde se combinan títulos, texto, imágenes y enlaces en una misma interfaz.

### Ejercicio 6. Secciones Semánticas Básicas
Organización del contenido utilizando etiquetas semánticas de HTML5 para mejorar la estructura y legibilidad del documento.

### Ejercicio 7. Listas Anidadas + Enlaces
Elaboración de una receta utilizando listas ordenadas, listas desordenadas y listas anidadas para organizar la información.

### Ejercicio 8. Tablas
Construcción de un horario de clases empleando tablas en HTML y estilos CSS para mejorar su presentación visual.

### Ejercicio 9. Formulario de Contacto
Diseño de un formulario con distintos tipos de campos para capturar información del usuario, acompañado de estilos personalizados.

### Ejercicio 10. Calculadora de Temperatura
Desarrollo de una aplicación sencilla que convierte grados Celsius a Fahrenheit mediante JavaScript, mostrando el resultado de forma inmediata en la página.

### Ejercicio 11. Convertidor de Distancias
Desarrollo de una calculadora que permite convertir distancias de kilómetros a millas mediante el uso de JavaScript y la captura de datos desde un formulario.

### Ejercicio 12. Convertidor de Divisas
Creación de una aplicación sencilla para convertir cantidades de Pesos Mexicanos (MXN) a Dólares Estadounidenses (USD), realizando el cálculo de manera automática.

### Ejercicio 13. Validación de Voto
Implementación de una validación donde el usuario ingresa su edad y el sistema determina si cuenta con la edad suficiente para ejercer su derecho al voto.

### Ejercicio 14. Ordenamiento de números
Desarrollo de una aplicación que trabaja con una lista de números para obtener el valor mayor, el valor menor y el promedio de los datos ingresados.

### Ejercicio 15. Registro de Estudiantes
Creación de un pequeño sistema para registrar estudiantes junto con sus calificaciones, permitiendo calcular el promedio del grupo e identificar las calificaciones más alta y más baja.

### Ejercicio 16. Calculadora de Operaciones Básicas
El objetivo del ejercicio es crear una calculadora web que permita al usuario realizar operaciones básicas (suma, resta, multiplicación y división) entre dos números ingresados.Para ello, se utilizarán funciones flecha en JavaScript.

### Ejercicio 17. Gestión de Tareas
Gestión de Tareas Pendientes con JSON, Local Storage, Funciones,Closures y Scope

### Ejercicio 18. Manipulación del DOM
Pequeña aplicación web que permite agregar y eliminar elementos de una lista dinámica.

## Objetivo

El propósito de esta actividad fue fortalecer los conocimientos básicos del desarrollo web mediante la creación de ejercicios prácticos que permitieran comprender el funcionamiento de la estructura de una página, el diseño con hojas de estilo y la interacción con JavaScript.
